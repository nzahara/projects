/********************************************************************
	Execution instructions
********************************************************************/

1. Install RabbitMQ Server using https://www.rabbitmq.com/install-windows.html.

2. Install Erlang using https://www.erlang.org/downloads.

3. Install Node.js using https://nodejs.org/en/.

4. Install MQSQL server.

5. Install Spring Tool Suite4.

6. Change mysql username and password in application.properties file present in the Java project.

7. Run Spring Boot Application.

8. Run Angular project using the commands mentioned below.


/********************************************************************
	Commands to Run Angular Project
********************************************************************/

1. npm install - To install missing libraries.
2. ng serve --open to start the angular application.
